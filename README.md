## About the Project

University team project developed within the WiWav student research group (SKN): smart greenhouse control system.

Responsible for the embedded/hardware part:
- FSM-based control logic for actuators (pump, lighting, heating, ventilation)
- sensor integration (temperature, humidity, soil moisture, pH, turbidity)
- custom serial protocol for communication with external application

The system periodically reads sensors, processes commands, and controls actuators based on time- and state-driven logic.

This repository contains a hardware-focused extract of my work.

Development period: Nov 2024 – Apr 2025.
